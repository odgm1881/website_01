Список изображений для одностраничного сайта. Все кадры — в едином арт-дирекшене бренда.

---

## Общий арт-дирекшен (для всех кадров)

- **Свет:** тёмный, тёплый, «при свечах»; глубокие мягкие тени. Без холодного и яркого света.
- **Палитра кадра = палитра бренда:** удовый коричневый, беж, крем, латунь. Без синевы, без кислотных цветов.
- **Фактуры:** дерево, латунь, ткань (лён/бархат), дым бахура, капля масла на коже, тиснёная бумага.
- **Подача:** макро-детали важнее общих планов. «Тихий люкс» — сдержанно, ремесленно, тактильно.
- **Стекло флакона:** матовое (frosted) кремовое или дымчато-янтарное — НЕ глянцевое прозрачное.
- **Крышки:** латунь или дерево. Никакого пластика «под золото», блёсток и металлик-градиентов.
- **Анти-гайд:** ❌ блестящее золото · ❌ мрамор и «дворцовый» китч · ❌ чистый чёрный/белый · ❌ кричащие цвета.

**Формат отдачи:** WebP или AVIF (+ JPG-исходник). Фоновые кадры — с запасом по ширине под ретину; в кадре оставлять затемнённые зоны под текст (поверх лежит тёплая виньетка).

## Как подключить готовые фото

1. Положить файлы в папку `images/` с именами из таблицы.
2. В `index.html` у каждого блока есть атрибут `data-img="..."` (например `data-img="rizk"`).
   Достаточно задать фон в `styles.css`, например:
   ```css
   [data-img="rizk"]  { background-image: url("images/rizk.jpg");  background-size: cover; background-position: center; }
   [data-img="hero"]  { background-image: url("images/hero.jpg");  background-size: cover; background-position: center; }
   ```
3. Убрать подсказки-плейсхолдеры одной строкой: `.img-note { display: none; }`
